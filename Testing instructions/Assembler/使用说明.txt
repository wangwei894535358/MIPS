Usage:   mips.exe XXX.asm

[ coding ]

taps由数字，大小写字母组成，只能且必须独占一行:
	j pos
	pos:
	add $1, $2, $3
指令操作符与操作码之间必须有空格，操作码之间逗号，隔开
注释写在行尾，由分号；开始

示例
	add $1, $2, $3	; this is add
	sub $1, $2, $3
	..
	..
	j aim		; jump to tap "aim"
	aim:		; this is a tap
	beq $1, $2, end	; if equal jump to end
	beq $1, $3, 3	; if $1==$3, jump to line=current+1+3
	end:
	j end
	nop		; do nothing
	nop		; do nothing
	j 0		; jump to the begining
